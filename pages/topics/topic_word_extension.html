어근과 접사, 합성어 등을 활용하여 낱말을 확장하는 다양한 방법을 배워봅시다.

홈 > 주제별 학습 > 낱말 확장하기

낱말 확장이란?

낱말 확장은 기존의 낱말을 활용하여 새로운 낱말을 만드는 과정입니다. 우리말에서는 주로 접사를 붙이는 파생어 만들기와 두 낱말을 합쳐서 만드는 합성어 만들기가 있습니다.

낱말 확장 방법을 알면 더 많은 어휘를 효과적으로 익히고 사용할 수 있어요. 또한 낯선 낱말을 만났을 때 그 구조를 분석하면 의미를 더 쉽게 파악할 수 있습니다.

파생어 만들기

파생어란?

파생어는 어근에 접사가 붙어서 만들어진 낱말입니다. 접사는 어근 앞에 붙는 접두사와 어근 뒤에 붙는 접미사로 나뉩니다.

접두사 사용하기
맨발 (맨 + 발)
새색시 (새 + 색시)
헛수고 (헛 + 수고)
덧붙이다 (덧 + 붙이다)
접두사는 어근의 앞에 붙어 새로운 의미를 더해줍니다. '맨'은 '덮이지 않은', '새'는 '새로운', '헛'은 '헛된', '덧'은 '더하여'의 의미를 더합니다.
접미사 사용하기
높이 (높 + 이)
구두쇠 (구두 + 쇠)
솜씨꾼 (솜씨 + 꾼)
늦장이 (늦 + 장이)
접미사는 어근의 뒤에 붙어 새로운 의미나 품사를 만들어줍니다. '이'는 명사화, '쇠'는 '도구', '꾼'은 '그 일을 하는 사람', '장이'는 '그러한 성질이 있는 사람'의 의미를 더합니다.
합성어 만들기

합성어란?

합성어는 두 개 이상의 낱말이 결합하여 만들어진 새로운 낱말입니다. 합성어는 각 낱말의 의미가 결합되어 새로운 의미를 만들어냅니다.

명사 + 명사
눈+사람 → 눈사람
밤+하늘 → 밤하늘
물+고기 → 물고기
두 명사가 결합하여 새로운 명사를 만듭니다. 각 명사의 의미가 결합되어 새로운 대상을 나타냅니다.
동사/형용사 어근 + 명사
쓰+레기 → 쓰레기
날+짐승 → 날짐승
맵+짜다 → 맵짜다
동사나 형용사의 어근이 명사와 결합하거나, 서로 결합하여 새로운 낱말을 만듭니다.

낱말 확장 방법 비교

파생어와 합성어를 비교하여 각각의 특징을 알아봅시다.

구분 파생어 합성어
정의 어근에 접사가 결합하여 만들어진 낱말 두 개 이상의 어근이 결합하여 만들어진 낱말
구성 요소 어근 + 접사(접두사/접미사) 어근 + 어근
예시 햇빛(햇+빛), 가르침(가르치+ㅁ), 꼬마(꼬+마) 밥그릇(밥+그릇), 빛나다(빛+나다), 산새(산+새)
특징 접사는 독립적으로 사용되지 않음 각 어근은 독립적으로 사용 가능
