# 초등 국어 문법 놀이터 - 홈페이지 디자인 세부 계획안

## 1. 웹사이트 구조 (사이트맵)

### 1.1 메인 구조
```
홈페이지
├── 학년별 학습 (1~6학년)
│   ├── 1학년
│   ├── 2학년
│   ├── 3학년
│   ├── 4학년
│   ├── 5학년
│   └── 6학년
├── 학습 주제별
│   ├── 문장의 기초
│   ├── 품사 탐험
│   ├── 문장 확장하기
│   └── 바른 표현
├── 학습 활동
│   ├── 게임
│   ├── 퀴즈
│   ├── 쓰기 연습
│   └── 말하기 연습
├── 학습 진도 관리
│   ├── 나의 학습 현황
│   ├── 학습 분석
│   └── 오답노트
├── 학부모/교사 공간
│   ├── 학습 모니터링
│   ├── 리포트
│   └── 학습 지원 자료
└── 고객 지원
    ├── 자주 묻는 질문
    ├── 이용 가이드
    └── 문의하기
```

### 1.2 상세 페이지 구성
- **홈 페이지**: 학습자 맞춤 추천, 최근 학습, 인기 콘텐츠, 공지사항
- **학년별 학습 페이지**: 해당 학년에 맞는 학습 주제 및 활동 모음
- **학습 콘텐츠 페이지**: 개별 학습 내용과 활동이 포함된 상호작용 페이지
- **게임/퀴즈 페이지**: 다양한 학습 게임과 퀴즈 활동을 제공하는 페이지
- **마이페이지**: 개인 학습 진도, 업적, 프로필 관리
- **학부모/교사 대시보드**: 학습자 활동 모니터링 및 분석 도구

## 2. 페이지별 레이아웃 및 디자인

### 2.1 메인 페이지 (홈)
#### 레이아웃 구성
```
+----------------------------------+
|             헤더                 |
|  로고  |  메뉴  |  검색  | 로그인|
+----------------------------------+
|                                  |
|         환영 배너/슬라이더       |
|                                  |
+----------------------------------+
|                                  |
|       학년별 빠른 이동 버튼      |
|                                  |
+----------------------------------+
|                                  |
|        추천 학습 콘텐츠          |
|                                  |
+----------------------------------+
|                                  |
|        나의 학습 현황            |
|                                  |
+----------------------------------+
|                                  |
|        인기 학습 활동            |
|                                  |
+----------------------------------+
|                                  |
|           푸터                   |
|                                  |
+----------------------------------+
```

#### 핵심 디자인 요소
- **환영 배너**: 밝고 활기찬 일러스트와 함께 학습 시작 버튼 배치
- **학년별 버튼**: 1~6학년 아이콘 형태의 직관적인 내비게이션 
- **추천 학습**: 카드 형태의 레이아웃으로 추천 콘텐츠 제시
- **학습 현황**: 시각적 진행 바와 차트로 학습 진도 표시
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 버전 레이아웃 최적화

### 2.2 학습 콘텐츠 페이지
#### 레이아웃 구성
```
+----------------------------------+
|             헤더                 |
+----------------------------------+
|                                  |
|       학습 주제 제목 및 설명     |
|                                  |
+----------------------------------+
|         |                        |
|         |                        |
|  내비   |                        |
|  게이션 |      학습 콘텐츠       |
|  메뉴   |      표시 영역         |
|         |                        |
|         |                        |
+----------------------------------+
|                                  |
|       학습 활동 선택 영역        |
|                                  |
+----------------------------------+
|                                  |
|    진행 현황 및 다음 버튼        |
|                                  |
+----------------------------------+
|                                  |
|           푸터                   |
|                                  |
+----------------------------------+
```

#### 핵심 디자인 요소
- **내비게이션 메뉴**: 현재 학습 주제 내 세부 목차 표시
- **콘텐츠 표시 영역**: 텍스트, 이미지, 애니메이션 등이 나타나는 영역
- **학습 활동 선택**: 해당 주제와 관련된 다양한 학습 활동 버튼 배치
- **진행 현황 바**: 현재 학습 진행도를 시각적으로 표시
- **다음/이전 버튼**: 명확하고 큰 크기의 내비게이션 버튼

#### 레이아웃 구성


#### 핵심 디자인 요소
- **몰입형 게임 영역**: 전체 화면에 가까운 크기로 집중도 향상
- **점수/진행 바**: 게임 진행 상황과 획득 점수 시각적 표시
- **인터랙티브 요소**: 드래그 앤 드롭, 클릭, 입력 등 다양한 상호작용
- **피드백 요소**: 정답/오답 시 시각/청각적 피드백 제공
- **힌트 버튼**: 어려움을 겪을 때 도움을 받을 수 있는 힌트 기능

- **타임라인**: 최근 학습 활동을 시간순으로 표시
- **배지 컬렉션**: 획득한 배지와 업적을 컬렉션 형태로 표시
- **추천 경로**: AI 기반 맞춤형 학습 추천 영역

## 3. 디자인 가이드라인

### 3.1 색상 팔레트
- **주요 색상**:
  - 메인 브랜드 색상: #4B7BEC (밝은 파란색) - 신뢰와 교육을 상징
  - 서브 색상: #FFA502 (밝은 주황색) - 창의성과 활력을 상징
  - 포인트 색상: #26DE81 (민트 그린) - 성취와 진행을 상징
  - 강조 색상: #FF4757 (선명한 빨간색) - 중요 알림과 경고에 사용

- **학년별 색상 코드**:
  - 1학년: #FF9FF3 (밝은 분홍색)
  - 2학년: #FECA57 (노란색)
  - 3학년: #FF6B6B (빨간색)
  - 4학년: #48DBFB (하늘색)
  - 5학년: #1DD1A1 (녹색)
  - 6학년: #5F27CD (보라색)

- **중립 색상**:
  - 배경: #F8F9FA (거의 흰색)
  - 텍스트 주요: #2C3E50 (짙은 남색)
  - 텍스트 보조: #95A5A6 (회색)
  - 구분선: #E0E7FF (밝은 회색)

### 3.2 타이포그래피
- **주요 서체**: 
  - 제목: Noto Sans KR Bold (또는 유사 무료 글꼴)
  - 본문: Noto Sans KR Regular
  - 강조: Noto Sans KR Medium

- **글자 크기**:
  - 메인 제목(h1): 32px (모바일: 24px)
  - 섹션 제목(h2): 24px (모바일: 20px)
  - 소제목(h3): 20px (모바일: 18px)
  - 본문 텍스트: 16px (모바일: 14px)
  - 보조 텍스트: 14px (모바일: 12px)
  - 버튼 텍스트: 18px (모바일: 16px)

- **줄 간격**: 1.5배
- **자간**: 0.5px

### 3.3 버튼 및 상호작용 요소
- **기본 버튼 스타일**:
  - 둥근 모서리 (border-radius: 12px)
  - 내부 여백 (padding: 12px 24px)
  - 호버/포커스 상태 명확히 표시
  - 그림자 효과로 입체감 부여

- **버튼 유형**:
  - 주요 버튼: 메인 색상 배경, 흰색 텍스트
  - 보조 버튼: 흰색 배경, 메인 색상 테두리와 텍스트
  - 비활성화 버튼: 회색 배경, 더 어두운 회색 텍스트

- **상호작용 요소**:
  - 클릭 가능 요소: 커서 변경 및 호버 효과
  - 드래그 요소: 손 모양 커서와 그림자 효과
  - 선택 요소: 명확한 선택 상태 표시

### 3.4 아이콘 및 일러스트레이션
- **아이콘 스타일**:
  - 둥근 모서리의 단순화된 선형 아이콘
  - 일관된 두께의 선 (2px 권장)
  - 학습 활동별 직관적인 아이콘 사용

- **일러스트레이션 가이드라인**:
  - 친근하고 귀여운 캐릭터 중심 디자인
  - 나이에 적합한 그림체 (저학년용 더 단순하고 밝은 스타일)
  - 포함 요소: 학습 가이드 캐릭터, 성취 배지, 장식적 요소

- **애니메이션 요소**:
  - 부드러운 전환 효과 (easing: ease-in-out)
  - 성취 시 축하 애니메이션
  - 페이지 로딩 애니메이션

## 4. 주요 페이지 상세 디자인

### 4.1 낱말 알아보기 (1학년) 페이지 상세
#### 콘텐츠 구성
1. **도입부**:
   - 낱말의 정의를 설명하는 친근한 캐릭터 애니메이션
   - "우리 주변의 말들, 낱말!" 제목 (큰 글씨와 밝은 색상)
   - 낱말에 대한 기본 설명 텍스트

2. **이름 낱말 섹션**:
   - 이름 낱말의 정의와 예시
   - 시각적 요소: 각 예시 낱말에 해당하는 귀여운 일러스트 (사과, 자동차, 학교 등)
   - 클릭하면 소리가 나오는 인터랙티브 요소

3. **동작 낱말 섹션**:
   - 동작 낱말의 정의와 예시
   - 애니메이션으로 표현된 동작 낱말 (달리다, 먹다, 자다 등)
   - 간단한 동작 애니메이션이 있는 카드 형태

4. **상태 낱말 섹션**:
   - 상태 낱말의 정의와 예시
   - 다양한 상태를 시각적으로 표현 (예쁘다, 크다, 작다 등)
   - 대비되는 상태를 비교하는 슬라이더 인터랙티브 요소

#### 학습 활동 디자인
1. **낱말 카드 매칭 게임**:
   - 화면 구성: 왼쪽에 단어, 오른쪽에 해당 이미지
   - 드래그 앤 드롭으로 단어와 이미지 연결
   - 정답 시 축하 애니메이션, 오답 시 재시도 안내
   - 난이도 조절 버튼 (쉬움/보통/어려움)

2. **낱말 분류하기 활동**:
   - 화면 구성: 상단에 분류할 낱말들, 하단에 '이름 낱말', '동작 낱말', '상태 낱말' 상자
   - 드래그 앤 드롭으로 낱말 카드를 해당 상자에 넣기
   - 분류 완료 후 결과 확인 및 설명 제공

3. **주변 낱말 찾기 활동**:
   - 현실적인 장면 일러스트 (교실, 놀이터, 집 등)
   - 장면 속 사물을 클릭하면 해당 낱말이 표시됨
   - 찾은 낱말 목록 자동 기록
   - 완료 후 나만의 낱말 사전으로 저장 가능

### 4.2 문장의 종류 배우기 (2학년) 페이지 상세
#### 콘텐츠 구성
1. **도입부**:
   - 다양한 표정의 캐릭터가 다양한 문장을 말하는 애니메이션
   - "문장의 종류 알아보기" 제목
   - 문장 종류의 중요성에 대한 짧은 설명

2. **평서문 섹션**:
   - 평서문의 정의와 용도
   - 다양한 평서문 예시 (말풍선 디자인)
   - 문장 끝 마침표(.) 강조 표시

3. **의문문 섹션**:
   - 의문문의 정의와 용도
   - 다양한 의문문 예시 (물음표 모양 말풍선)
   - 의문사(누가, 언제, 어디서 등) 하이라이트

4. **명령문 & 청유문 섹션**:
   - 각 문장 유형의 정의와 차이점
   - 시각적 비교를 통한 설명
   - 상황별 사용 예시 일러스트

5. **감탄문 섹션**:
   - 감탄문의 정의와 특징
   - 감정 표현과 연결된 예시
   - 감탄사 강조 표시

#### 학습 활동 디자인
1. **문장 종류 맞추기 퀴즈**:
   - 화면에 문장이 표시됨
   - 4개의 문장 종류 버튼 중 선택
   - 연속 정답 시 난이도 자동 상승
   - 결과 그래프로 이해도 시각화

2. **상황에 맞는 문장 만들기**:
   - 다양한 상황 일러스트 제시 (축하, 질문, 요청 등)
   - 해당 상황에 맞는 문장 유형 선택
   - 직접 문장 만들어 입력하기
   - 문장 저장 및 공유 기능

3. **문장 변환 게임**:
   - 주어진 평서문을 다른 종류의 문장으로 바꾸기
   - 단계별로 난이도 상승
   - 시간제한 모드 옵션
   - 변환된 문장 음성으로 들어보기 기능

### 4.3 학부모/교사 대시보드 페이지
#### 콘텐츠 구성
1. **대시보드 요약**:
   - 학습자 이름, 학년, 최근 활동 요약
   - 학습 시간, 완료 활동 수, 정답률 등 주요 지표
   - 최근 학습 내용 캘린더 형태 표시

2. **학습 진도 섹션**:
   - 교과 과정 기반 진도 표시

## 5. 반응형 디자인 가이드라인

### 5.1 디바이스별 최적화
- **데스크톱 (1024px 이상)**:
  - 풀 기능 인터페이스
  - 멀티 컬럼 레이아웃
  - 호버 효과 활용
  - 고해상도 일러스트레이션

- **태블릿 (768px ~ 1023px)**:
  - 간소화된 내비게이션 (햄버거 메뉴 옵션)
  - 2-3 컬럼 레이아웃
  - 터치 최적화 인터페이스
  - 일부 기능 간소화

- **모바일 (767px 이하)**:
  - 단일 컬럼 레이아웃
  - 햄버거 메뉴 내비게이션
  - 터치 중심 인터페이스
  - 콘텐츠 우선순위 재배치
  - 핵심 기능만 표시

### 5.2 터치 인터페이스 고려사항
- **터치 타겟 크기**: 최소 44px x 44px
- **스와이프 제스처**: 페이지 전환, 카드 넘기기에 활용
- **핀치 제스처**: 확대/축소 기능 (필요시)
- **탭 피드백**: 시각적 + 진동 피드백 제공

## 6. 접근성 가이드라인

### 6.1 접근성 고려사항
- **색상 대비**: WCAG 2.1 AA 표준 준수 (최소 4.5:1)
- **키보드 내비게이션**: 모든 기능 키보드로 접근 가능
- **스크린 리더 호환성**: 적절한 alt 텍스트 및 ARIA 레이블
- **텍스트 크기 조정**: 확대/축소 기능 지원
- **애니메이션 제어**: 움직임 줄이기 옵션 제공

### 6.2 특수 필요 지원
- **읽기 지원**: 텍스트 음성 변환 기능
- **집중력 지원**: 단순화된 인터페이스 모드
- **시각적 지원**: 고대비 모드, 글꼴 크기 조정
- **청각적 지원**: 자막 및 시각적 피드백

## 7. 상호작용 및 애니메이션 디자인

### 7.1 마이크로 인터랙션
- **버튼 호버/탭**: 부드러운 색상 변화 및 약간의 확대
- **드래그 앤 드롭**: 그림자와 투명도로 드래그 상태 표시
- **입력 필드**: 포커스 상태 명확히 표시
- **완료 상태**: 체크마크 애니메이션으로 완료 강조

### 7.2 트랜지션 효과
- **페이지 전환**: 페이드 인/아웃 또는 슬라이드 효과
- **콘텐츠 로딩**: 스켈레톤 화면 또는 단계적 표시
- **모달 창**: 배경 흐림 효과와 함께 부드럽게 표시
- **내비게이션 변경**: 부드러운 재배치 애니메이션

### 7.3 학습 동기부여 애니메이션
- **성취 축하**: 화려한 색종이/별 애니메이션
- **레벨업/배지 획득**: 빛나는 효과와 함께 확대 애니메이션
- **진행 상황**: 채워지는 애니메이션으로 진도 시각화
- **캐릭터 리액션**: 학습자 활동에 반응하는 캐릭터 표정/동작

## 8. 기술적 구현 고려사항

### 8.1 프론트엔드 기술
- **프레임워크**: React.js 기반 컴포넌트 구조
- **스타일링**: CSS Modules 또는 Styled Components
- **애니메이션**: GSAP 또는 Framer Motion 라이브러리
- **상호작용**: React DnD(드래그 앤 드롭), React Spring(물리 기반 애니메이션)

### 8.2 성능 최적화
- **이미지 최적화**: WebP 형식, 적절한 크기 조정, 지연 로딩
- **코드 분할**: 라우트/컴포넌트 기반 분할로 초기 로딩 시간 단축
- **애니메이션 최적화**: requestAnimationFrame, 하드웨어 가속 활용
- **오프라인 지원**: 주요 콘텐츠 및 활동의 PWA/서비스 워커 구현

### 8.3 개발 우선순위
1. 핵심 학습 경로 및 컨텐츠
2. 게임 및 인터랙티브 요소
3. 학습 진도 추적 시스템
4. 학부모/교사 대시보드
5. 고급 기능 (AI 추천, 심화 분석 등)
